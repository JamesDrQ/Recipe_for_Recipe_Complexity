Recipe Ratings Prediction Project

Introduction

This project analyzes a recipes dataset and explores the relationship between recipe features and average user ratings.

Question

I investigate whether recipe complexity, such as the number of ingredients, is related to average recipe rating.

Hypothesis Test

I tested whether recipes with many ingredients and recipes with few ingredients have different average ratings.

Prediction Problem

I plan to predict average recipe rating using recipe-level features such as minutes, number of ingredients, number of steps, and nutrition information.
